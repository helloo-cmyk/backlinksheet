## PART 3 — STYLE TAB DEEP DIVE

- **Presets:** 4 pills (Brat, Deluxe/Black, Crash/Pink, White).
- **Background Color:** Hex picker, controls `state.bg`. Default `#c1ff00`.
- **Text Color:** Hex picker, controls `state.fg`. Default `#0a0a0a`.
- **Contrast Check:** Logic in `contrastRatio()` compares BG and FG for accessibility.
- **Aspect Ratio:** Dropdown, controls `state.ratio`.
- **Resolution:** Dropdown, controls `state.res`. Default `1500`.
- **Selfie Mode:** File input `#brat-bgUpload` (accepts images) and `#brat-clearBg` button.

---

## PART 4 — STICKERS TAB DEEP DIVE

- **Content:** 5 Categories (Love, Faces, Sparkles, Cute, Symbols).
- **Stickers:** 37 emojis total (rendered as bitmaps for canvas performance).
- **Interaction:** Stickers can be dragged, resized (corners), and rotated (top handle).
- **Image Upload:** NO image upload in this tab. Image upload is currently only for **Backgrounds** (Style Tab).

## PART 7 — CANVAS BEHAVIOR

- **Engine:** Vanilla HTML5 Canvas (No Fabric.js or Konva).
- **Text Rendering:** Custom `drawText()` function handles wrapping, character-by-character kerning, and `state.textTransform` (drag/rotate/scale).
- **Sticker Rendering:** `drawStickers()` renders cached emoji bitmaps.
- **Live Update:** All inputs use `input` or `change` listeners that trigger `requestDraw()` (wrapped in `requestAnimationFrame`).
- **Max Length:** No hard limit enforced in JS, but UI box wrapping makes very long text unreadable.
